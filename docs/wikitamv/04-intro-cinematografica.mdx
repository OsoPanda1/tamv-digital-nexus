# Intro Cinematográfica

## Visión General

La intro cinematográfica de TAMV MD-X4™ es una experiencia inmersiva que combina:
- **Motor visual 3D** con Three.js
- **Máquina de estados** para control de flujo
- **Sistema de partículas** cuánticas
- **Audio inmersivo** sincronizado
- **Shaders personalizados** para efectos visuales

### Especificación de Estados

```typescript
type Phase =
  | 'permission'   // S0: Idle - Black screen + accept button
  | 'awakening'    // S1: Activation - Dark pulse / heartbeat
  | 'ignition'     // S2: Origin_Render - Power surge
  | 'explosion'    // S3: Infrastructure_Projection - Quantum big bang
  | 'cosmos'       // S4: Dedication_Anchor - Stars fly
  | 'crown'        // S5: Global_Expansion - Leadership moment
  | 'logo'         // S6: Transition_To_Core - Logo reveal
  | 'message'      // Final: Isabella narrative
  | 'done';        // Complete
```

- Pantalla negra con ambiente sutil

## Sistema de Partículas Cuánticas

### QuantumField Component

```typescript
function QuantumField({ phase }: { phase: Phase }) {
  const COUNT = 12000;
  
  // Posiciones iniciales en esfera
  for (let i = 0; i < COUNT; i++) {
    const theta = Math.random() * Math.PI * 2;
    const phi = Math.acos(2 * Math.random() - 1);
    const r = 0.3 + Math.random() * 3.5;
    
    positions[i] = {
      x: r * Math.sin(phi) * Math.cos(theta),
      y: r * Math.sin(phi) * Math.sin(theta),
      z: r * Math.cos(phi)
    };
  }
  
  // Velocidades para efecto de vuelo
  const speed = flying ? 0.32 : 0.015;
  
  return (
    <Points positions={positions} stride={3}>
      <PointMaterial
        vertexColors
        size={0.045}
        transparent
        opacity={0.92}
        blending={AdditiveBlending}
      />
    </Points>
  );
}
```

### PowerCore Geometry

```typescript
function PowerCore({ phase }: { phase: Phase }) {
  // Escala dinámica según fase
  const targetScale = {

      <TorusRing radius={2.1} color="#FF2D78" />

}
```

## Leadership Crown

### Geometría de la Corona

```typescript
function LeadershipCrown({ phase }: { phase: Phase }) {
  const SPIKE_ANGLES = [0, 60, 120, 180, 240, 300];
  
  return (
    <group>
      {/* Base ring */}
      <mesh rotation={[Math.PI / 2, 0, 0]}>
        <torusGeometry args={[1.1, 0.09, 16, 80]} />
        <meshStandardMaterial color="#FFD700" emissive="#FFD700" />
      </mesh>
      
      {/* Picos alternados */}
      {SPIKE_ANGLES.map((angle, i) => (
        <mesh
          key={i}
          position={[
            Math.cos(angle * Math.PI / 180) * 1.1,
            0.3 + (i % 2 === 0 ? 0.55 : 0.2),
            Math.sin(angle * Math.PI / 180) * 1.1
          ]}
        >
          <coneGeometry args={[0.08, i % 2 === 0 ? 0.9 : 0.5, 8]} />
          <meshStandardMaterial color="#FFD700" emissive="#FFD700" />
        </mesh>
      ))}
      
      {/* Gema central */}
      <mesh position={[0, 0.3, 0]}>
        <octahedronGeometry args={[0.28, 0]} />
        <meshStandardMaterial color="#00D9FF" emissive="#00D9FF" />
      </mesh>
    </group>
  );
}
```
